## An R Package for Coral Reef Ecosystem Modeling

MizerReef is an extension of the mizer package for modeling size-structured fish communities, specifically adapted for coral reef ecosystems.

### Features

- Size-structured multispecies modeling for coral reefs
- Integration with coral reef habitat complexity data
- Fisheries management scenario testing
- Climate change impact assessment

### Installation

```r
# Install from GitHub
devtools::install_github("cmbeese/MizerReef")
```
